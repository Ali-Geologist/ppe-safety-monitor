## Deployment

This app is optimized for Streamlit Cloud deployment:

1. **Fork this repository**
2. **Go to [Streamlit Cloud](https://share.streamlit.io)**
3. **Connect your GitHub account**
4. **Select this repository**
5. **Set main file path to `app.py`**
6. **Click "Deploy"**

## Usage

1. **Project Setup**: Configure safety requirements for your operation
2. **Camera Setup**: Add IP cameras or use test mode
3. **Monitoring**: Start live safety monitoring
4. **Dashboard**: View compliance analytics
5. **Reports**: Generate safety compliance reports
